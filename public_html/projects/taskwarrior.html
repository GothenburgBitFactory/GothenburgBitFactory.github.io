About

Originally named task, the project began as a demo C++ port of the todo.sh program in November 2006. This was so that features could be added that were not easily done using shell scripts, such as text formatting and color rules.

Since 2006, taskwarrior has been used in the development of ... taskwarrior. Dogfood.

The demo needed autoconf, portability, testing and documentation, and when combined with a year of inactivity, meant version 1.0.0 was not released until June 2008.

Since then, more than twenty releases have been made. In that time, taskwarrior has been incorporated into 10+ Linux distros and Cygwin, gained man pages, a wiki, and a bug list. Well over 400 bugs have been logged and fixed, and 300 feature requests have been addressed.

There is an active IRC channel and forum. Over 100 persons have contributed to the project.

Testing

Taskwarrior undergoes hourly automated testing. This involves running the test suite of over 5,000 unit tests.

Future

Coming up soon are features like synchronization with the task server, user-defined attributes, and report listing breaks.
