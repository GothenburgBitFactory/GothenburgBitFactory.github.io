libkronisk

0.8.0 Development

Kronisk is a date/time library. It will soon replace Taskwarrior's date/time parsing and formatting code. It supports the parsing of strings to time stamps, and the formatting of time stamps to strings. It provides a C API, for maximum utility.

libkronisk will be a required dependency of future Taskwarrior releases.

  • git clone: git://tasktools.org/kronisk.git

About

Taskwarrior has long supported date parsing and formatting using custom date formats. A natural enhancement to that would be to support ISO-8601:2004, which is a global standard for representing dates, times, durations and recurrences.

When taskwarrior began to support 'easter' as a date, it became clear that supporting date expressions (eom + 2wks - 2days, or 3rd thursday in November) was desirable.

When taskwarrior began shipping holiday files, it became clear that maintaining these was a burden, and automation was the solution. Then the website holidata.net was created to do just that, but still it needed manual data generation and verification.

Libkronisk has been created to supply both taskwarrior and holidata.net with powerful date handling capabilities. As a library it will provide useful functionality to any program.
