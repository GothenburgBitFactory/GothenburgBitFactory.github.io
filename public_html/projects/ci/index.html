About

Ci is an open source, distributed continuous integration system, but not in the traditional sense. It employs agents to perform build, test and other functions. It is very low friction, and nearly invisible.

Ci is designed to augment single- or multi-platform software projects by providing a build and test framework that automatically builds and tests projects without any intervention beyond a single command to set up the system.
