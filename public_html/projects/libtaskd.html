libtaskd

1.0.0.alpha Alpha

git clone git://tasktools.org/libtaskd.git

libtaskd is a client-side library that accesses the Taskserver. It implements the encrypted communication with the Taskserver using GnuTLS.

About

libtaskd will be an optional dependency of future taskwarrior releases.

Development

The libtaskd library will provide a simple interface to the Task Server that manages sync operations.

In conjunction with taskwarrior and taskd, libtaskd will be made available when functional.
