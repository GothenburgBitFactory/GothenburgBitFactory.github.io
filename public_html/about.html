tasktools

Tasktools is a collection of technologies and products that provide support for the activities surrounding Taskwarrior development, testing and distribution.

There are many projects developed and hosted on tasktools.org.

 

The technologies currently in use are:
  • Git for source management
  • Atlassian Stash for git hosting and repository management
  • Atlassian Jira for issue tracking
  • nginx HTTP and mail proxy server
  • Apache HTTP server
  • Perl for unit testing
  • Python for unit testing
  • RackSpace hosting
  • PiWik for analytics
Services generously provided by:
  • RackSpace hosting
  • Pingdom monitoring
