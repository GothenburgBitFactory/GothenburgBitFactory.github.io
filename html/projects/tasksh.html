Tasksh

Tasksh is a shell command that wraps Taskwarrior commands. It is intended to provide simpler Taskwarrior access, and add a few new features of its own. The project is new, bear with us please.

Tasksh replaces the built-in shell command of older releases, and the bundled tasksh program of version 2.3.0. The former was very limited and the latter unsupported, buggy and flawed. This project represents GBF taking control of this project.

  • git clone https://git.tasktools.org/scm/ex/tasksh.git
  • brew install tasksh
  • Flod testing 1.0.0

About

Tasksh is (so far) a bare-bones shell that exists as a separate program, in a separate code repository, on it's own release schedule that is unhampered by lengthy Taskwarrior development cycles.

Development

Future Tasksh releases are being planned now. Features currently being designed include:

  • Persistent Context
  • Pomodoro Timer
  • Task Review
  • Notifications
